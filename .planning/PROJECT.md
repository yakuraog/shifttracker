# ShiftTracker — Автозаполнение таблицы смен из Telegram

## What This Is

Система автоматического заполнения таблицы смен на основе фотографий из Telegram-групп. Бот мониторит рабочие чаты охранных предприятий, распознает факт заступления сотрудника на смену по фото с подписью и автоматически проставляет отметку "1" в таблице смен. Рассчитана на масштаб до 200 групп с тысячами фото в сутки.

## Core Value

Автоматическое и достоверное заполнение таблицы смен без ручного участия оператора — каждая отметка подтверждена фотографией из рабочего чата.

## Current Milestone: v1.0 MVP

**Goal:** Рабочая система, которая принимает фото из тестовых Telegram-групп, определяет сотрудника по простым правилам и ставит "1" в таблицу смен.

**Target features:**
- Telegram-бот для мониторинга групп и приема фотографий
- Идентификация сотрудника по подписи/Telegram-аккаунту
- Определение даты смены с учетом временных окон
- Автоматическое заполнение Google Sheets
- Журнал обработки сообщений
- Обработка спорных случаев (ручная проверка)
- Административный интерфейс для настройки

## Context

- Домен: охранные предприятия, учет выходов на смену
- Масштаб: до 200 Telegram-групп, тысячи фото/сутки в каждой
- Существующий процесс: ручное заполнение таблицы смен
- Триггер отметки: фото сотрудника с подписью в момент заступления
- В каждой группе уже есть или будет Telegram-бот
- Тестовое задание — нужна чистая архитектура и работающий прототип

## Constraints

- **Stack**: Python 3.11+, aiogram 3, FastAPI, PostgreSQL, SQLAlchemy — выбрано для оптимальной работы с Telegram Bot API и async
- **Telegram API**: Bot API (не MTProto) — боты должны быть добавлены в группы с правами чтения сообщений
- **Таблица смен**: Google Sheets API — наиболее вероятный формат
- **Performance**: Near real-time обработка, очередь для пиковых нагрузок
- **Масштаб v1**: Ограниченное число тестовых групп

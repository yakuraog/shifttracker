   ```python
   import gspread
   from google.oauth2.service_account import Credentials

   SCOPES = ["https://www.googleapis.com/auth/spreadsheets"]

   def build_client(credentials_file: str) -> gspread.Client:
       creds = Credentials.from_service_account_file(credentials_file, scopes=SCOPES)
       return gspread.authorize(creds)
   ```
   This is a thin factory. The writer will call it once at startup.

3. Create `shifttracker/sheets/header_cache.py`:
   - Module-level `_cache: dict[tuple[str, str], tuple[list[str], float]]` (key = (spreadsheet_id, sheet_name), value = (headers, timestamp from `time.monotonic()`))
   - `CACHE_TTL = 300` (5 minutes, integer seconds)
   - `get_cached(spreadsheet_id: str, sheet_name: str) -> list[str] | None` — returns headers if cache hit and not expired, else None
   - `set_cached(spreadsheet_id: str, sheet_name: str, headers: list[str]) -> None` — stores headers with current monotonic time
   - `invalidate(spreadsheet_id: str, sheet_name: str) -> None` — removes entry
   - `clear_all() -> None` — clears entire cache (useful for tests)
   - Use `time.monotonic()` not `datetime.utcnow()` — monotonic clock is not affected by system clock changes.

4. Create `shifttracker/sheets/cell_resolve.py`:
   ```python
   import gspread.utils
   from datetime import date

   def resolve_cell(
       header_row: list[str],
       sheet_row: int,
       shift_date: date,
   ) -> tuple[str, str] | None:
       """Resolve (value_cell_a1, link_cell_a1) or None if date column not found.

       Tries both zero-padded DD.MM and non-padded D.M formats against the header.
       Returns A1 addresses for the value cell and the adjacent link cell (col+1).
       """
       if not sheet_row or sheet_row < 1:
           return None

       # Try multiple date formats for header matching
       candidates = [
           shift_date.strftime("%d.%m"),         # "09.04"
           f"{shift_date.day}.{shift_date.month}",  # "9.4" (cross-platform)
           shift_date.strftime("%d.%m.%Y"),      # "09.04.2026"
           f"{shift_date.day}.{shift_date.month}.{shift_date.year}",  # "9.4.2026"
       ]

       for col_idx, header in enumerate(header_row, start=1):
           stripped = header.strip()
           if stripped in candidates:
               value_cell = gspread.utils.rowcol_to_a1(sheet_row, col_idx)
               link_cell = gspread.utils.rowcol_to_a1(sheet_row, col_idx + 1)
               return value_cell, link_cell
       return None
   ```

1. Create `shifttracker/pipeline/stages/shift_date.py`:
   ```python
   from datetime import date, datetime, timedelta
   from zoneinfo import ZoneInfo
   from typing import Optional

   TOLERANCE_HOURS = 2

   def resolve_shift_date(
       message_datetime: datetime,
       shift_start_hour: int,
       shift_end_hour: int,
       timezone: str = "Europe/Moscow",
   ) -> tuple[Optional[date], Optional[str]]:
   ```

   Algorithm:
   a. Convert `message_datetime` to local time in the given timezone using `zoneinfo.ZoneInfo`.
   b. Extract `local_hour = local_dt.hour`, `local_date = local_dt.date()`.
   c. Determine if this is a **night shift window**: `is_night_shift = shift_start_hour > shift_end_hour` (e.g. 22 > 6).
   d. **Day shift** (start < end, e.g. 06-22):
      - If `shift_start_hour <= local_hour < shift_end_hour`: return `(local_date, None)` — within window.
      - If `local_hour < shift_start_hour` and `shift_start_hour - local_hour <= TOLERANCE_HOURS`: return `(local_date, None)` — within pre-start tolerance.
      - If `local_hour >= shift_end_hour` and `local_hour - shift_end_hour < TOLERANCE_HOURS`: return `(local_date, None)` — within post-end tolerance.
      - Otherwise: return `(None, "outside_time_window")`.
   e. **Night shift** (start > end, e.g. 22-06):
      - If `local_hour >= shift_start_hour`: return `(local_date, None)` — evening part of night shift, date = today.
      - If `local_hour < shift_end_hour`: return `(local_date - timedelta(days=1), None)` — morning part, date = yesterday.
      - If `shift_start_hour - local_hour <= TOLERANCE_HOURS and local_hour >= shift_end_hour`: return `(local_date, None)` — within pre-start tolerance.
      - If `local_hour >= shift_end_hour and local_hour < shift_start_hour - TOLERANCE_HOURS`: return `(None, "outside_time_window")`.
